
配置 `cypress.config.js`：
在项目根目录下创建或编辑 `cypress.config.js` 文件，添加以下内容：
const { defineConfig } = require("cypress");
const cypressHtmlReporter = require('cypress-html-reporter');

module.exports = defineConfig({
  e2e: {
    baseUrl: 'http://localhost:8080',
    specPattern: 'cypress/e2e/**/*.cy.{js,jsx,ts,tsx}',
    setupNodeEvents(on, config) {
      cypressHtmlReporter(on, config);
      return config;
    },
    // 其他配置项
  },
});

运行以下命令启动服务器：

   ```bash
   http-server -p 8080
   ```
打开 Cypress 测试运行器：

```bash
npx cypress open
```
您也可以在命令行中运行所有测试：

```bash
npx cypress run
```

批量运行 F:\course\engineering\2025-group-24\tests\cypress\e2e\Feb21\ 目录下的所有测试并生成报告:  npx cypress run --spec 'cypress/e2e/Feb21/**/*.cy.js'

将执行指定的测试文件，并在测试完成后自动生成 HTML 格式的测试报告。通常会生成在项目的默认reports目录中。要查看报告，可以在浏览器中访问 `http://localhost:8080` 查看生成的测试报告。
